# Channel Multiply

Production url https://channelmultiply.surge.sh 

## TL;DR

Launch project in a container

```sh
docker-compose build --no-cache
docker-compose up -d     
docker ps # pick the id of canal-multiply
docker exec -it <id> sh 
```

Launch directly

```sh
yarn
yarn dev
```
